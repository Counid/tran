# tran
this program is aimed for easy to use translate myanmar in termux

# Install
$pkg install git\
$git clone https://github.com/Counid/tran.git \
$chmod +x *\
$cd\
$./setupTran

# use
type 'tran' to start use
press enter to clear
eg :  words :he
